# Extensions - Métodos de Extensão do AppHost

Extension methods para simplificar a configuração da infraestrutura do MeAjudaAi.

## Arquivos

- **KeycloakExtensions.cs** - Métodos de extensão para Keycloak
- **PostgreSqlExtensions.cs** - Métodos de extensão para PostgreSQL
- **MigrationExtensions.cs** - Métodos de extensão para migrations automáticas

## Classes Relacionadas

- **Options/** (raiz do AppHost) - Classes de configuração
- **Results/** (raiz do AppHost) - Classes de resultado
- **Services/** (raiz do AppHost) - Hosted services (ex: MigrationHostedService)
